## 1. 实验目的

&emsp;&emsp;（1）理解单周期CPU工作过程；

&emsp;&emsp;（2）理解指令存储器和数据存储器的哈佛结构存储；

&emsp;&emsp;（4）掌握单周期CPU设计与实现方法。
